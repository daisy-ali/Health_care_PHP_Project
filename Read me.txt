# HOW TO GET STARTED

Open your XAMPP Control Panel and start Apache and MySQL.

Extract the downloaded source code zip file.

Copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.

Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin

Create a new database naming health.

Import the provided SQL file. The file is known as DATABASE health.sql located inside the Database folder.

Browse the Doctor's Appointment Systsem in a browser. i.e. http://localhost/Ali-Hassan-php-project/.
